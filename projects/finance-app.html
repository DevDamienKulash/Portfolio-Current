Finance App

A full-stack financial tracking application that allows users to monitor income, expenses, and overall financial trends in real time. Designed to simplify personal budgeting through dynamic data visualization and clean UI structure.

Key Features

  • Add / Edit / Delete transactions
  • Category breakdown charts
  • Monthly filtering
  • Real-time total calculations
  • Responsive dashboard layout

Technical Breakdown

  • Built with React and component-based architecture
  • Managed application state using React Hooks
  • Implemented dynamic rendering based on transaction data
  • Integrated chart visualization library for analytics
  • Structured reusable components for scalability
